## Method 1: GitHub Pages (Recommended - 100% Free)

### Step 1: Create GitHub Account
1. Go to [github.com](https://github.com)
2. Sign up for free account
3. Verify your email

### Step 2: Create Repository
1. Click "New Repository"
2. Name it: `sparkle-jewels` (or any name you like)
3. Make it **Public** (required for free GitHub Pages)
4. Click "Create Repository"

### Step 3: Upload Files
1. Click "uploading an existing file"
2. Drag and drop ALL project files:
   - index.html
   - admin.html
   - order-tracking.html
   - styles.css
   - admin.css
   - script.js
   - admin.js
   - README.md
3. Write commit message: "Initial website upload"
4. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. Go to repository "Settings" tab
2. Scroll down to "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Select "main" branch
5. Click "Save"

### Step 5: Get Your Live URL
- Your website will be live at: `https://yourusername.github.io/sparkle-jewels`
- Wait 5-10 minutes for deployment
- Access admin panel: `https://yourusername.github.io/sparkle-jewels/admin.html`

## Method 3: Vercel (Developer Friendly)

1. Visit [vercel.com](https://vercel.com)

### Step 2: Deploy
1. Click "New Project"
2. Import your GitHub repository
3. Click "Deploy"
4. Your site will be live at: `https://sparkle-jewels.vercel.app`
